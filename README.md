# 🚗 MB Plus Benidorm
 
> A modern, full-stack car dealership web application built with Next.js and Java Spring Boot — featuring dynamic inventory management, image uploads, and a clean customer-facing storefront.
>
> [![Live Demo](https://img.shields.io/badge/Live%20Demo-mbplusbenidorm.es-blue?style=for-the-badge&logo=vercel)](https://www.mbplusbenidorm.es/)
 
---

## 📸 Overview
 
MB Plus Benidorm is a production-ready dealership platform that lets staff manage vehicle listings while customers browse inventory in a polished, responsive interface. The project demonstrates end-to-end full-stack development across a TypeScript/React frontend and a RESTful Java backend, backed by cloud-native infrastructure.
 
---
 
## 🛠️ Tech Stack
 
### Frontend
| Technology | Purpose |
|------------|---------|
| Next.js 14 | React framework with SSR & App Router |
| TypeScript | Type-safe development |
| Tailwind CSS | Utility-first styling |
| Cloudinary | Image upload, storage & optimization |
 
### Backend
| Technology | Purpose |
|------------|---------|
| Java (Spring Boot) | RESTful API server |
| Supabase | Managed PostgreSQL database |
| Hibernate / JPA | ORM & database access layer |
